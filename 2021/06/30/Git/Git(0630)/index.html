Git-分支

分支

  • 一開始就有一個分支
  • master： （像片片貼紙）指著某個Commmit
  • Head: （平常跟在某個分支旁邊）
  • git branch 檢視目前分支

切換分之的意思是 要做的時候切到

操作

  • 快捷鍵： git checkout [-q] [-f] [-m]
  • 特殊快捷鍵：git checkout. =>檔案全部救回來
