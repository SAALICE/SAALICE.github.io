程式語言的鄙視鏈

  • 例子：設計師與美工差別，前端工程師和切版員差別，C#和C井（sharp）差別， python3 與 python2差別（3與2版本匯出的方法不同）

  • VanillaJS . larver. TIOBE

  • 所有工程師都鄙視php工程師

  • 習得 python ruby 技能 > php(基礎)

  • Q:一個人要學多少語言比較好
    A:適合自己的

  • 寫得好，能解決問題的才是好的工程師。AI的背後都是數學和大數據

  • 建議工程師每年都可以學習一個新語言

  • 新手刷題重要嗎（可保持頭腦清晰），大公司偏向邏輯問題、演算法

  • 新手刷題網：codewords

  • （重要）新手面試問題：如果這個DEMO PJ一個人重做 你會要做多久 （這是一個態度 跟自信）

  • 那些厲害的程式語言或工具，當年是怎麼被發明出來的

  • Ruby / JS /GIT （說明背景如何來） ＧＯ

  • 可試試研究、GO語言

  • RUBY是什麼語言寫的 :都是C語言做出來的/

  • 程式語言都是先有底 才有才有辦法寫

  • 會寫程式的工程師

  • 如果一個作品是程式語言，你根本不需要說明你的作品。
