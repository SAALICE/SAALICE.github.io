< Rails （專案）>

專案復刻對象：Evernote 功能

rails環境安裝

$which ruby
$which rails
(確認以上兩者都有rvm字樣)
$rails -v (gem install rails)
$which yarn(前端工具開發用)
$which node
$node -v >14
準備好的話，就rails new 新專案。

建立新專案： rails new 新檔名

啟動伺服器： rails s

開啟瀏覽器： localhost:3000

資料與資料庫建立

$rails generate(建立) scaffol（鷹架）＋內容（自串可省）

migrate:

  • rockback (危險)
  • 新增或修正內容：直接rails g db:migrate (再開新描述檔)
  • shema:load （修復檔案）

＊new :只有他是新增 其他都不是
＊token ：預設機制

rails g controller notes

rails g model Note title

(✏️慣例式是命名)

Resource 資源：8條路徑 / 手動新增路徑

新增路徑：

從路徑考量去決定使用何種方法。
兩者差別在有無需要id，可視為多個或、單個較情況使用。

GET /notes 筆記列表
GET /notes/3 看第三頁筆記
GET /notes/new 建立新列表
GET /notes/edit 編輯列表
post /notes 寫入筆記
PUT /notes/3 更新3號筆記
PUT /notes/3edit 編輯3號筆記
DELETE /

GET/PUT/POST/DELETE :針對不同內如的不同動作
註：（字串等都會被自動轉：varchar）

建立筆記中的超連結 ：creat new
建立表格：送出資料
建立資料：form_for
